# Playwright Automation Framework

This project is built using Playwright with JavaScript and follows the Page Object Model (POM) design pattern.

## Features

- Login Automation Test
- Invalid Login Test
- Add To Cart Test
- Remove Product From Cart Test
- Checkout Flow Test
- Page Object Model (POM)
- Assertions and Validations
- beforeEach Hooks

## Technologies Used

- Playwright
- JavaScript
- Page Object Model (POM)
- Node.js

## Project Structure

playwright-automation/

pages/
- LoginPage.js
- CartPage.js

tests/
- LoginPom.spec.js
- CartPom.spec.js
- checkout.spec.js

## How to Run Tests

Install dependencies:

npm install

Run tests:

npx playwright test

Run tests in headed mode:

npx playwright test --headed
