# Playwright Automation Framework

Automation Testing Framework built using Playwright with JavaScript and Page Object Model (POM).

## Overview

This project demonstrates end-to-end UI automation and API testing using Playwright. The framework follows the Page Object Model (POM) design pattern for better code reusability and maintainability.

## Technologies Used

* Playwright
* JavaScript
* Node.js
* Page Object Model (POM)
* Git & GitHub

## UI Test Scenarios Covered

### Authentication

* Valid Login
* Invalid Login Validation
* Logout Functionality

### Cart & Checkout

* Add Product To Cart
* Remove Product From Cart
* Checkout Flow
* Order Validation

### Playwright Concepts Practiced

* Locators
* Assertions
* Hooks (beforeEach)
* Wait Handling
* New Tab Handling
* Hover Actions
* Double Click Actions
* Dropdown Handling
* Iframe Handling
* Conditional Statements (If Else)
* Loops

## API Testing

The framework includes API automation practice using Playwright APIRequestContext.

* GET Request
* POST Request
* PUT Request
* DELETE Request

## Framework Design

* Page Object Model (POM)
* Reusable Methods
* Separate Page and Test Layers
* Maintainable Project Structure
