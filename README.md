# Playwright Automation Framework

This project is built using Playwright with JavaScript and follows the Page Object Model (POM) design pattern. The framework covers UI Automation Testing, API Testing, SQL Practice, Postman Collections, and Jenkins CI Integration.

## Project Overview

The framework was created to practice and demonstrate automation testing skills including:

* UI Automation using Playwright
* API Testing using Playwright and Postman
* SQL Query Practice
* Jenkins Continuous Integration (CI)
* GitHub Version Control
* Page Object Model (POM) Framework Design

---

## Framework Features

## How to Run Tests

Install Dependencies:

npm install

Run All Tests:

npx playwright test

Run Tests in Headed Mode:

npx playwright test --headed

Generate HTML Report:

npx playwright show-report
